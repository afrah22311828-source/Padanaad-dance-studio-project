Padanaad Dance Studio Application
Overview

Padanaad Dance Studio Application is a mobile-based platform designed to modernize Bharatanatyam learning and dance class management. The application provides a centralized system where teachers, students, and parents can interact, manage activities, and access learning resources efficiently.

The platform replaces traditional manual processes with a structured digital solution, improving accessibility, transparency, and overall user experience.

Features
User Management
User registration and login
Role-based access (Teacher, Student, Parent)
Secure authentication system
Dashboard
Personalized dashboards for each user
Quick access to attendance, fees, content, and announcements
Content Management
Teachers can upload videos, notes, and session summaries
Students can access learning materials anytime
Attendance System
Teachers can mark attendance
Students and parents can view attendance records
Fee Management
Track paid and pending fees
Secure online payment system
Automatic receipt generation
Communication System
Unified chat interface
Real-time messaging between users

System Architecture

The application follows a Client–Server Architecture:

Frontend (Mobile App) handles user interaction
Backend processes business logic
API layer connects frontend and backend
Database stores user, attendance, and payment data
Cloud storage manages media files
Installation & Setup
Clone the repository
git clone https://github.com/your-username/padanaad-dance-app.git
Navigate to project folder
cd padanaad-dance-app
Install dependencies
npm install
Start the application
npx expo start
